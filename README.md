# Spotify UI Clone 🎧

This project is a simple Spotify UI Clone created for learning frontend development.

## 🚀 Features

* Spotify inspired user interface
* Music player layout
* Responsive design
* Play/Pause UI controls

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Purpose

This project was created to practice frontend development and UI design by recreating the interface of Spotify.

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.
